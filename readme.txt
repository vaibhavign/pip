This plugin adds order invoice & packaging list printing support to WooCommerce.

== Installation ==

1. Upload the plugin folder to the '/wp-content/plugins/' directory.

2. Activate the plugin  through the 'Plugins' menu in WordPress.

== Usage ==

Go to WooCommerce > PIP Settings, configure and start printing. Simple!

== Custom templates ==

1. Copy woocommerce-pip-template directory to the '/wp-content/themes/yourtheme/woocommerce/'
   directory.
   
2. Edit the templates as you wish.
